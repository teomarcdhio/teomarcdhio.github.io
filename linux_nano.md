### Nano commands

Delete ( or cut ) lines  
```ctrl + k```   
Past cutted lines   
```ctrl + u```   
Search for text   
```ctrl + w```   
Find and Replace   
```alt + r```   
Saving Your Work Without Exiting nano   
```ctrl + o```  
Displaying the Current Line Number   
```alt + c```
